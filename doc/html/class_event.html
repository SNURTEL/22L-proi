Detailed Description

Represents a single event of type Event::EventType.

Event types are represented with EventType enum. EventType always needs to be passes to constructor, followed by appropriate args. If incorrect args are provided, an EvenConstructionException is thrown.

Event class has multiple constructor that can be used for initializing different events.

Events can be generated by any classed and should be passed to an event queue (./EventQueue.h)
